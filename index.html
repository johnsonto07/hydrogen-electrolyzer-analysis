Water Electrolysis · Hydrogen Generation

Finding the safest and most efficient operating conditions for hydrogen production.

This project investigates how applied voltage and current affect hydrogen output in a low-cost water electrolysis setup. The goal is to compare gas production, energy use, and overall efficiency while keeping the process organized, measurable, and safety-aware.

View Data View Graphs

Project Goals

  • Measure hydrogen production across multiple voltage settings
  • Track current, time, and gas volume collected
  • Compare energy input against hydrogen output
  • Identify a practical voltage range that balances safety and efficiency

Overview

About the experiment

The setup is designed to electrolyze water using a simple benchtop apparatus with electrodes, a power source, and gas collection containers. By testing different operating conditions, the project aims to understand how electrical input changes hydrogen production rate and system performance.

This page is built so you can keep everything for the project in one place: photos, raw data, graph images, and interactive charts generated directly from your CSV data file.

Main Question

Which operating conditions produce hydrogen effectively without pushing the setup beyond a reasonable safety margin?

Variables to Track

Voltage, current, run time, electrolyte concentration, hydrogen volume, oxygen volume, and temperature.

Outputs

Interactive plots, graph image uploads, raw data tables, and a clean project summary suitable for presentation.

Recommended Use
